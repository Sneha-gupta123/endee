# AI Grocery Semantic Search using Endee

## Project Overview
This project is an AI powered grocery search application that allows users to find grocery items across multiple stores using semantic search and vector similarity.

The system uses embeddings and vector similarity search to retrieve relevant grocery products and provide recommendations.

## Problem Statement
Users often struggle to find the best grocery products across multiple stores and compare prices quickly. This project demonstrates how vector databases and semantic search can improve product discovery.

## Features
- Semantic grocery search
- Multi store product comparison
- AI based recommendations
- Shopping cart functionality
- Voice search support
- Modern UI built with Streamlit

## System Architecture
User Query → Embedding → Vector Search → Product Retrieval → AI Recommendation → UI Display

## Tech Stack
- Python
- Streamlit
- Vector similarity search
- Embeddings
- CSV dataset
- Endee vector database (reference architecture)

## How Endee is Used
Endee is used as the conceptual vector database architecture for managing embeddings and enabling high speed similarity search for semantic product retrieval.
